# Portus，支持书签管理的 [Tab Harbor](https://github.com/V-IOLE-T/tab-harbor) 分叉版本

[English](README.md) | [简体中文](README.zh-CN.md)

**支持书签管理的 [Tab Harbor](https://github.com/V-IOLE-T/tab-harbor) 分叉版本。**

Portus 将你的 Chrome 新标签页转变为一个冷静、有序的“安静桌面”工作区。它将打开的标签页、手动分组、快捷链接、稍后阅读、轻量待办和集成书签功能结合在一个本地优先的界面中。

## 书签支持

Portus 将浏览器书签直接集成到“安静桌面”工作区中，让你无需离开新标签页即可无缝访问和整理书签库。

- **层级导航**：通过清晰的文件夹视图和直观的面包屑导航浏览整个书签库。
- **拖拽重排**：通过将项目拖动到新位置，按照你喜欢的方式组织书签。
- **移动到文件夹**：通过将书签或文件夹拖动到面包屑或其他文件夹图标上，轻松实现移动和归类。
- **集成管理**：一键在新标签页中打开书签，或删除书签（包括整个文件夹树），并带有安全确认。
- **一键收藏**：点击控制台中的按钮，即可将任何快捷链接（Quick Link）快速保存到浏览器书签中。
- **设计风格统一**：书签视图遵循原项目简约风的设计美学，支持主题色适配和流畅的交互动画。

## 💻 安装

1. 克隆此分支：

   ```bash
   git clone -b feat/bookmark https://github.com/jhcao1024/tab-harbor
   ```

2. 打开 `chrome://extensions`
3. 开启 **Developer mode**
4. 点击 **Load unpacked**
5. 选择 [`extension/`](extension/) 文件夹
6. 打开一个新标签页
